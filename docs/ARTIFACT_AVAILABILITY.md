# Artifact Availability

## Public artifacts available now

The following artifacts are intentionally public in this repository today:

- schema documentation
- anonymized public release tables at approximately 1% of paper scale
- benchmark JSON schemas
- the `public_release` benchmark tier with split files and public references
- slim public reproducibility scripts for validation, submission checks, and release-level statistics
- anonymization policy and redaction dictionary
- manifests, checksums, and release-level flag definitions

## Planned larger release

The following artifacts are planned for an expanded post-publication release, subject to compliance review:

- larger or full schema-stable data tables
- official full benchmark instances and candidate sets
- expanded reference labels and group bundles
- additional provenance hooks
- richer versioning and maintenance metadata

## Principle

The repository should stay precise about what is public **today**. The goal is a high-quality public release, not ambiguous wording that makes the current anonymized release sound like the full artifact.
