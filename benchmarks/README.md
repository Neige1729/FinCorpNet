# Public benchmark assets

This directory contains the public benchmark surface for FinCorpNet.

It includes:

- submission schemas for Task A and Task B
- the `public_release/` benchmark tier for the anonymized 1%-scale release
- public reference files that can be checked with the slim scripts under `scripts/`

The public release tier is the reviewable benchmark layer in this upload package. Expanded full-scale benchmark bundles remain subject to post-publication compliance review.
